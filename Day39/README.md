# 📊 Day 39 – Excel Coding Challenge

## 🗳️ Election Data Analysis Dashboard

---

## 📌 Project Overview

In this project, I analyzed a synthetic dataset of a recent state election in India using **Microsoft Excel**.
The objective was to extract insights on voting patterns, party performance, and demographic trends.

---

## 📂 Dataset Details

The dataset contains:

* Constituency
* State
* Party
* Candidate
* Votes
* Total Votes
* Gender
* Age

✔️ Total Records: **125 rows**
✔️ Tool Used: **Microsoft Excel**

---

## 🧹 Task 1: Data Cleaning

* Removed duplicate records
* Ensured correct data types (Votes, Age as numeric)
* Validated total votes per constituency

---

## 📊 Task 2: Vote Share Calculation

Created a new column:

```
Vote % = (Votes / Total Votes) * 100
```

---

## 🏆 Task 3: Winners Identification

Identified winning candidates in each constituency using:

* **MAXIFS()**
* **IF()**
* Pivot Table (Top 1 filter)

✔️ Extracted:

* Candidate Name
* Party
* Votes

---

## 📈 Task 4: Party Performance

### 🔹 Total Votes by Party:

* **XYZ** → 2,116,608 🥇
* **ABC** → 1,984,883
* **PQR** → 558,542

### 🔹 Insight:

* XYZ received the highest vote share (~45%)
* ABC closely followed (~43%)
* PQR had significantly lower support

---

## 🗳️ Constituency Winners Summary

* **XYZ** → Won majority of constituencies 🥇
* **ABC** → Won few constituencies
* **PQR** → No wins

✔️ Insight:
XYZ performed consistently across constituencies.

---

## 👥 Task 5: Gender Analysis

### 🔹 Total Votes:

* Female → 2,360,208 (~50.6%)
* Male → 2,299,825 (~49.4%)

### 🔹 Insight:

* Slight **female dominance**
* Overall voting is **balanced**

---

## 🎂 Task 6: Age Analysis

* Average Age → **~37 years**
* Youngest Candidate → **32 years**
* Oldest Candidate → **55 years**

### 🔹 Insight:

* Majority winners fall in **30–40 age group**
* Mix of youth and experience

---

## 📊 Task 7: Dashboard Creation

### ✔️ Dashboard Components:

* 📊 Bar Chart → Party vs Total Votes
* 🥧 Pie Chart → Gender Vote Share
* 🏆 Table → Winners per Constituency

### ✔️ Features:

* Clean layout
* Interactive slicers (Party, Gender, Constituency)

---

## 🚀 Key Insights

* XYZ party dominated in constituencies due to consistent performance
* ABC remained a strong competitor
* Female candidates slightly outperformed male candidates
* Younger candidates (30–40) showed higher success rate
