#  Day 12 – MGNREGS Scheme Analysis (Power BI)

##  Project Overview

This project analyzes the performance of the MGNREGS across different states and years using Power BI. The goal is to evaluate efficiency, identify gaps, and provide data-driven recommendations.

---

##  Objectives

* Analyze employment demand vs supply
* Measure scheme efficiency across states
* Identify high and low performing regions
* Forecast future trends
* Simulate budget impact using What-if analysis

---

##  Dataset

The dataset includes:

* Households Demanded Work
* Households Provided Work
* Person-Days Generated
* Expenditure
* Wages Paid

---

##  Data Preparation

* Cleaned missing values and inconsistencies
* Standardized year format
* Created calculated measures for analysis

---

##  Key Metrics (DAX)

* Work Completion Rate
* Expenditure per Person-Day
* Average Wage
* Total Person-Days
* Total Expenditure

---

##  Dashboard Features

###  Trend Analysis

* Person-days generated over time
* Expenditure growth trends

###  State-Level Comparison

* Top and bottom performing states
* Efficiency comparison

###  Geographic Analysis

* Map showing employment distribution

###  Advanced Analytics

* Forecasting future trends
* What-if parameter for budget simulation
* Decomposition Tree for root cause analysis
* Key Influencers visual

---

##  Key Insights

* Person-days show a steady increase, indicating rising reliance on the scheme
* Significant variation exists across states in performance and efficiency
* High expenditure does not always guarantee better outcomes
* Payment delays negatively impact worker participation
* Efficient states maintain balance between demand and execution

---

##  Recommendations

* Improve fund utilization efficiency
* Reduce payment delays through automation
* Focus on underperforming states
* Use predictive analytics for planning
* Allocate funds based on performance metrics

---

## Limitations

* No district-level data available
* Synthetic dataset used for analysis
* Limited external socio-economic factors

---

## Conclusion

This dashboard provides a comprehensive view of MGNREGS performance, helping policymakers make informed decisions to improve rural employment outcomes.
