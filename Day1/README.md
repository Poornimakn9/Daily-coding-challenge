### **Day 1: Excel Data Analysis Task**

### **Objective**
Analyze the given sales dataset using Microsoft Excel.

### **Dataset**
The dataset contains sales data of different products across multiple regions and subcategories.
Columns included:
- Order ID
- Order Date
- Product
- Region
- Sub-category
- Quantity
- Unit Price

### **Task Performed**

###  Data Cleaning
- Checked for missing or inconsistent values
- Ensured proper formatting of dates and numbers

###  Revenue Calculation
- Created a new column: **Total Revenue**
- Used formula:
  
  Total Revenue = Quantity × Unit Price

###  Pivot Table Analysis
Created Pivot Tables to analyze:

- Total Revenue by Region
- Total Revenue by Product

###  Top Performing Products
- Identified the **Top 5 products** based on revenue

### Visualization
- Created a chart to visualize **Revenue by Region**

### **Key Insights**
-  West region generated the highest revenue
-  Desk Chair was the highest-selling product
-  Electronics category produced the highest revenue

##  How to View
Open the Excel file (.xlsx) using Microsoft Excel or Google Sheets.
