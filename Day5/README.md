### **Day 5: IT Help Desk Analysis**

### **Objective**
Analyze help desk ticket data using Microsoft Excel to identify trends, resolution performance, and recurring issues.

### **Dataset**
The dataset contains IT support tickets with the following columns:
Ticket ID
Submission Date
Resolution Date
Category
Priority
Assigned Analyst
Description

### **Task Performed** 

### Data Cleaning

Checked for missing values
Ensured proper date formatting
Verified consistency in categories and priority

### Resolution Time Calculation
Created a new column: Resolution Time (Hours)
Resolution Time = (Resolution Date - Submission Date) × 24

### Added extra columns:
Day of Week
Month
Hour

### Analysis Performed
### Ticket Trends
Analyzed ticket count by:
Month
Day of Week
Hour

### Resolution Time Analysis

Compared average resolution time by:
Category
Priority

### Analyst Performance

Evaluated performance based on:
Average resolution time
Number of tickets handled

### Root Cause Analysis

Identified frequently occurring issues from descriptions

### Visualization

Created pivot tables for all analysis
Built charts for:
Ticket trends
Revenue (Resolution insights)
Analyst performance

### Key Insights

July and August had the highest number of tickets
Crashing issues occurred most frequently
Password-related issues are common
Resolution time varies across analysts
Tickets are distributed across working hours

### Recommendations

Automate password reset process
Investigate frequent crashing issues
Improve support during peak months
Provide training for analysts with higher resolution time
