**Day 14 – SQL Sales Analysis (Coding Challenge)**

On Day 14, I worked on a Sales dataset to practice fundamental SQL concepts like filtering, sorting, and data retrieval. This challenge helped strengthen my understanding of real-world data queries.

**Dataset Overview**
The dataset contains sales transaction details including:

Customer Name
City
Product Name
Category
Quantity
Price
Sale Date

**Key Concepts Practiced**
SELECT statements
Filtering using WHERE
Sorting using ORDER BY
Removing duplicates using DISTINCT
Conditional queries on numeric and text data

**Tasks Performed**
Retrieved all records from the table
Displayed specific columns like customer name and city
Filtered sales by city (e.g., Bangalore)
Extracted products from a specific category (Electronics)
Applied conditions on price and quantity
Sorted data (price low → high, quantity high → low)
Identified unique cities

**Key Learnings**
Learned how to filter data efficiently using conditions
Understood the importance of sorting for analysis
Gained clarity on handling duplicates using DISTINCT
Built a strong foundation for advanced SQL queries
