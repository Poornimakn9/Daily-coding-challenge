**Day 21 – Insurance Claims Analysis (Subqueries)**

This project focuses on solving real-world SQL problems using subqueries on an insurance claims dataset.

**Key Tasks Covered**
Identified claims exceeding the overall average claim amount
Extracted policy holders with approved claims
Compared individual claims with policy holder averages
Displayed claims alongside overall average claim value

**Skills Used**
Subqueries (Single-row & Correlated)
Aggregate Functions (AVG)
Filtering with WHERE and IN
Real-world data analysis approach

**Outcome**
Improved understanding of subqueries and analytical thinking in SQL, essential for Data Analyst roles.
