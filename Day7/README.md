## Day7-Quality Control Dashboard (Power BI)

This project focuses on analyzing manufacturing production data to monitor product quality and identify defects.

### Key Features:
- Calculated Overall Defect Rate using DAX
- Identified Top 5 Defective Products
- Analyzed Defect Trends over Time
- Evaluated Performance by Production Line and Inspector
- Interactive dashboard with Date slicer

### Tools Used:
- Power BI
- DAX
- Data Modeling

### Outcome:
The dashboard helps production managers quickly identify quality issues and take corrective actions.
